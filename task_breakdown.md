# Tensionote 开发任务拆分

## 1. 目标

本任务拆分面向 Tensionote V1，覆盖：

- iOS 原生开发
- Android 原生开发
- 共享产品逻辑对齐

## 2. 阶段划分

### 阶段 1：基础搭建

- 建立 iOS 工程目录
- 建立 Android 工程目录
- 建立模型层
- 建立本地化资源结构
- 建立设计系统常量

### 阶段 2：首页与记录核心流程

- 首页快速录入
- 首页最近 2 周趋势图
- 直接状态说明
- 详细录入页
- 本地保存记录

### 阶段 3：历史与趋势

- 历史记录列表
- 记录详情
- 记录编辑
- 趋势详情页

### 阶段 4：提醒、报告、设置

- 本地提醒
- PDF 报告
- 邮件分享
- 语言切换
- 隐私和免责声明页面

### 阶段 5：质量和发布

- 三语言校对
- 异常规则测试
- 多设备适配
- 上架准备

## 3. iOS 任务

### 3.1 基础架构

- 建立 SwiftUI App 入口
- 建立 AppRouter
- 建立 ViewModel 基类和状态流转规范
- 建立 Repository 协议

### 3.2 首页

- 首页快速录入布局
- 输入校验
- 立即保存逻辑
- 跳转详细录入页
- 2 周趋势区
- 点选记录显示直接状态说明

### 3.3 记录模块

- 记录实体
- 本地存储
- 详细录入页
- 编辑页
- 详情页

### 3.4 历史模块

- 历史列表
- 时间倒序查询
- 直接状态文案显示

### 3.5 趋势模块

- 14 天默认趋势
- 7/14/30 天切换
- 范围带绘制
- 异常点高亮

### 3.6 提醒与报告

- 通知权限申请
- 本地通知调度
- PDF 生成
- 邮件分享

### 3.7 设置

- 跟随系统语言
- 手动语言切换
- 合规页面

## 4. Android 任务

### 4.1 基础架构

- 建立 Compose App 入口
- 建立导航图
- 建立 ViewModel 和 StateFlow
- 建立 Repository 接口

### 4.2 首页

- 首页快速录入布局
- 输入校验
- 立即保存逻辑
- 跳转详细录入页
- 2 周趋势区
- 点选记录显示直接状态说明

### 4.3 记录模块

- 记录实体
- Room 数据库存储
- 详细录入页
- 编辑页
- 详情页

### 4.4 历史模块

- 历史列表
- 时间倒序查询
- 直接状态文案显示

### 4.5 趋势模块

- 14 天默认趋势
- 7/14/30 天切换
- 范围带绘制
- 异常点高亮

### 4.6 提醒与报告

- 通知权限和提醒注册
- AlarmManager 调度
- PDF 生成
- 邮件 Intent 分享

### 4.7 设置

- 跟随系统语言
- 手动语言切换
- 合规页面

## 5. 共享逻辑任务

- 定义血压记录模型
- 定义提醒模型
- 定义异常规则输出格式
- 定义三语言状态文案 key
- 定义首页聚合数据结构

## 6. 当前优先级

P0：

- 产品名统一
- 首页快速录入
- 本地记录保存
- 最近 2 周趋势
- 直接状态说明

P1：

- 历史记录
- 记录详情/编辑
- 趋势详情

P2：

- 提醒
- 报告
- 邮件分享
- 设置页
